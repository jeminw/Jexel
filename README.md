# Jexel
A modern, futuristic pixel-style font inspired by technology, gaming, and cyberpunk culture.

## Overview
Jexel is a sleek pixel font designed for tech-forward creative projects. Combining clean geometric shapes with a sharp digital aesthetic, it’s perfect for:
- Logos & branding
- UI/UX design
- Game interfaces
- Posters & digital art
- Tech product packaging

## Key Features
- Retro-futuristic pixel style
- High readability across digital/print
- Minimalist geometric structure
- Cyberpunk & tech-inspired visual language
